# AttendAI Project Context

## Project

AttendAI is an AI-powered Attendance Verification System for educational
institutions.

Teachers upload an Excel sheet containing absent students.

The system automatically calls students using an AI voice agent,
collects the reason for absence, categorizes it, generates summaries,
and prepares downloadable reports.
