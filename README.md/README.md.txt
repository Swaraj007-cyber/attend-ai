# AttendAI

AI-powered Attendance Verification System for Educational Institutions.

## Overview

AttendAI automates the process of verifying student absences.

Teachers upload an Excel file containing absent students. The system uses an AI voice agent to call each student, collect the reason for absence, and generate intelligent reports.
