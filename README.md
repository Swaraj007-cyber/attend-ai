# 🎓 AttendAI

> AI-Powered Attendance Verification System for Educational Institutions

AttendAI is an intelligent SaaS platform that automates the process of verifying student absences using AI voice agents.

Instead of manually calling absent students, teachers simply upload an Excel sheet containing the names and phone numbers of absent students. AttendAI automatically contacts each student, conducts a natural AI-powered conversation in their preferred language, understands the reason for absence, and generates a detailed report for the institution.

---

# ✨ Features

- 📂 Upload Excel sheets of absent students
- 🤖 AI Voice Agent calls every student automatically
- 🌍 Supports multiple languages
- 🧠 AI understands and summarizes absence reasons
- 📊 Interactive dashboard with attendance analytics
- 📄 Generate Excel reports with collected reasons
- 📈 Daily attendance insights and statistics
- 🔍 Search, filter, and export call records
- 🔐 Secure institute login and authentication
- ☁️ Cloud-based SaaS platform

---

# 🚀 Vision

AttendAI aims to eliminate the manual effort involved in attendance follow-up while improving communication between educational institutions and students through intelligent AI automation.
